JSON in PowerShell-Objekte umwandeln

JSON ist ein Standardformat für APIs und Konfigurationsdaten. ConvertFrom-Json macht daraus direkt ein nutzbares PowerShell-Objekt.

Beispiel: Meeting-Daten auslesen

PowerShell
$data = Invoke-WebRequest https://psugh.github.io/meeting-data.json | ConvertFrom-Json $data.nextMeeting.date

Das gibt den Termin als String zurück (z. B. Freitag, den 13.03.2026, ab 18:30h).

Auf Objekt-Eigenschaften zugreifen

PowerShell
$data.nextMeeting.topics | ForEach-Object { $_.title }

JSON wieder erzeugen

PowerShell
$data | ConvertTo-Json -Depth 5

Wenn du die JSON-Struktur manipulieren willst, kannst du Eigenschaften hinzufügen oder ändern und dann mit ConvertTo-Json zurückschreiben.
